每个组件的this 是一样的吗？是什么?

为什么可以结构赋值得到，data 中的数据

```
data(){
    return {
      username:'',
      password:'',
      userId:''
    }
  },
  methods:{
    login(){
      let { username,password } = this
    }     
```

结论：

是不一样的，每个组件的this 代表当前组件，_uid 是标识

会拥有 data 中的属性，和 method 中的方法，可以结构出来

有 $parent 代表父组件

$root 代表根组件



比如 login 组件的 uid 是4， 父组件 uid 是3， 根组件 uid 是2

uid 不是固定的，是单调递增的，所以每次不一样
